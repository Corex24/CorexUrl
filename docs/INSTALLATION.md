# Installation Guide

## Prerequisites
- Node.js 16 or higher
- npm or yarn package manager

## Quick Start

### Via npm
```bash
npm install corex
```

### Via yarn
```bash
yarn add corex
```

### Via pnpm
```bash
pnpm add corex
```

## Installation from GitHub
For the latest development version:

```bash
npm install github:Corex24/Corex
```

## Verify Installation
After installation, you can verify it worked by checking:

```bash
npm list corex
```
